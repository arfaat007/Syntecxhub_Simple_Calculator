Simple calculator 

description:
This is a first project assigned as task 1 by **syntecxhub techno;ogy** for python programming internship,
It allows user to perform basic arithmetic operations as instructed and it also supports clear and exit options for easy interactions.

Features:
perform basic arithmetic: +,-,/,*
handles invalid inputs
prevents divide-by-zero errors
allows repeated calculations using loop
Simple text-based interface

How It Works:

The program asks you to enter:
First number
Operator (+, -, *, /)
Second number

It performs the operation and shows the result.
You can type:
clear → to reset the calculator
exit → to quit the program

How to Run:
Save the program as calculator.py
Open your terminal or VS Code terminal
Run the program using:
python calculator.py

Requirements:
Python 3.x installed on your system
Works on any OS (Windows, macOS, Linux)
